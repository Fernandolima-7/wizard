Backup do Banco de Dados

Crie uma cópia de segurança completa do banco de dados SQLite. O arquivo será salvo automaticamente com data e hora.

Sobre os backups

  • Os arquivos são salvos automaticamente com timestamp no nome
  • Formato: wizard-backup-YYYY-MM-DDTHH-mm-ss-sssZ.db
  • Recomenda-se realizar backup antes de qualquer alteração importante
  • Para restaurar: selecione um backup na lista abaixo ou substitua manualmente o arquivo prisma/wizard.db
